### 安装使用步骤 📔

```text
git clone https://github.com/lixaviers/xavier-pms.git
```

- **Install：**

```text
npm install
```

- **Run：**

```text
npm run dev
```

- 请使用 npm，不推荐 yarn、cnpm
- 已设置国内镜像源(加速)
